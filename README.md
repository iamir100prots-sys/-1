1.1 Наименование программы
SiteBlocker (Блокировщик сайтов)

1.2 Назначение программы
Программа предназначена для временной блокировки доступа к веб-сайтам на компьютере под управлением ОС Windows путём создания правил исходящей блокировки в брандмауэре Windows.

1.3 Целевая аудитория
Пользователи, желающие ограничить доступ к определённым сайтам на ограниченный период

Системные администраторы для временного ограничения доступа

Родительский контроль

1.4 Требования к операционной системе
Windows 7 и выше

Наличие брандмауэра Windows

Права администратора для запуска


2. Функциональные требования
2.1 Основные функции
2.1.1 Блокировка сайтов
ID	Требование	Приоритет
F-01	Программа должна принимать на вход одно или несколько доменных имён через пробел	Высокий
F-02	Программа должна автоматически удалять из введённой строки протоколы (http://, https://)	Средний
F-03	Программа должна удалять завершающие слеши (/) из доменного имени	Средний
F-04	Программа должна автоматически блокировать как вариант с www, так и без www	Высокий
F-05	Программа должна выполнять DNS-разрешение домена для получения всех IP-адресов	Высокий
F-06	Программа должна создавать правила в брандмауэре Windows для каждого IP-адреса	Высокий
F-07	Программа должна блокировать сайт на 24 часа (86400 секунд)	Высокий
F-08	Программа должна проверять, не заблокирован ли сайт уже, перед повторной блокировкой	Высокий
2.1.2 Разблокировка сайтов
ID	Требование	Приоритет
F-09	Программа должна обеспечивать ручную разблокировку сайта по доменному имени	Высокий
F-10	Программа должна автоматически удалять правила брандмауэра при разблокировке	Высокий
F-11	Программа должна удалять сайт из внутреннего списка блокировок	Высокий
F-12	Программа должна уведомлять пользователя об успешной разблокировке	Низкий
2.1.3 Автоматическая разблокировка
ID	Требование	Приоритет
F-13	Программа должна иметь фоновый поток для автоматической проверки истечения времени блокировки	Высокий
F-14	Фоновый поток должен проверять блокировки каждые 60 секунд	Высокий
F-15	При истечении 24 часов программа должна автоматически удалять правило брандмауэра	Высокий
F-16	Фоновый поток должен корректно завершаться при выходе из программы	Высокий
2.1.4 Управление списком блокировок
ID	Требование	Приоритет

F-18	Программа должна показывать для каждого сайта: домен, время блокировки, оставшееся время, список IP	Средний
F-19	Программа должна сохранять список блокировок в файл blocked_sites.dat	Высокий
F-20	Программа должна загружать сохранённый список блокировок при запуске	Высокий
F-21	При запуске программа должна очищать список от истёкших блокировок	Высокий
2.1.5 Пользовательский интерфейс
ID	Требование	Приоритет
F-22	Программа должна предоставлять консольное меню с выбором действий	Высокий
