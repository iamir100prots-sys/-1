1.1 Наименование программы
SiteBlocker (Блокировщик сайтов)

1.2 Назначение программы
Программа предназначена для временной блокировки доступа к веб-сайтам на компьютере под управлением ОС Windows путём создания правил исходящей блокировки в брандмауэре Windows.

1.3 Целевая аудитория
Пользователи, желающие ограничить доступ к определённым сайтам на ограниченный период

Системные администраторы для временного ограничения доступа

Родительский контроль

1.4 Требования к операционной системе
Windows 7 и выше

Наличие брандмауэра Windows

Права администратора для запуска


2. Основные функции
Блокировка сайтов
Программа принимает одно или несколько доменных имён через пробел. Автоматически удаляет протоколы http:// и https://, убирает завершающие слеши. Блокирует оба варианта домена — с www и без www. Выполняет DNS-разрешение для получения всех IP-адресов сайта. Создаёт правила в брандмауэре Windows для каждого IP-адреса. Проверяет, не заблокирован ли сайт уже, перед повторной блокировкой.

Разблокировка сайтов
Программа позволяет вручную разблокировать сайт по доменному имени. При разблокировке удаляются все правила брандмауэра для этого сайта, запись удаляется из внутреннего списка, выводится уведомление пользователю.

Автоматическая разблокировка
Фоновый поток проверяет список заблокированных сайтов каждые 60 секунд. При истечении 24 часов автоматически удаляет правило брандмауэра. Поток корректно завершается при выходе из программы.

Управление списком
Программа отображает список всех заблокированных сайтов с указанием домена, времени блокировки, оставшегося времени и списка IP-адресов. Список сохраняется в файл blocked_sites.dat и загружается при запуске. При старте программа очищает истёкшие блокировки.

Пользовательский интерфейс
Консольное меню с выбором действий: 1-блокировка, 2-список, 3-разблокировка, 4-выход. Поддерживается ввод и вывод кириллицы в кодировке UTF-8. Временные метки выводятся в формате ГГГГ-ММ-ДД ЧЧ:ММ:СС.



3. Требования к безопасности
Программа проверяет наличие прав администратора при запуске. При отсутствии прав выводится сообщение об ошибке и программа завершается. Все команды брандмауэра выполняются с правами администратора.

4. Требования к производительности и надёжности
Время DNS-разрешения одного домена зависит от сети. Фоновый поток использует Sleep и не нагружает процессор. Программа корректно обрабатывает повреждённый файл данных. При ошибке создания правила для одного IP остальные IP обрабатываются. Корректно обрабатывается ситуация с недоступными DNS-серверами.

5. Потокобезопасность
Доступ к глобальному списку блокировок синхронизируется с помощью мьютекса. Фоновый поток не блокирует основной поток надолго. Флаг остановки программы является атомарным.

6. Совместимость
Программа работает с IPv4 и IPv6 адресами. Поддерживаются Windows 7, 8, 10 и 11. Используются только стандартные библиотеки, дополнительные зависимости не требуются.

7. Технический стек
Язык программирования — C++11 и выше. Сетевая подсистема — Winsock2. Управление брандмауэром — Netsh advfirewall. Кодировка — UTF-8. Многопоточность реализована через std::thread, std::mutex и std::atomic.

Используемые библиотеки: windows.h, ws2tcpip.h, iostream, fstream, sstream, string, vector, ctime, thread, mutex, atomic, algorithm.
